# My Python Learning Journey
This repository contains my early Python projects, focusing on fundamental concepts, automation, and data handling.
   ## Project List:
* **first.py**: Introduction to variables, user input, and basic conditional logic.
* **goal_tracker.py**: A financial goal tracker using `while` loops and error handling (`try-except`).
* **list_practice.py**: Managing inventory using Python lists and list methods.
* **search_logic.py**: An optimized search tool for inventory management with input formatting.
* **price_list.py**: Using dictionaries to store and retrieve product pricing data.
* **file_reader.py**: Automating data retrieval from external `.txt` files.
* **file_writer.py**: An interactive tool for saving and appending data to files.
